# UrbanPulse Adventures — Static Demo

This repository now hosts a polished static demo of the UrbanPulse Adventures site using plain HTML, CSS, and JavaScript for quick local preview and easy sharing.

Run locally

- Serve the demo with the included npm script:

```bash
npm install   # only if you need to install `http-server` or other tools
npm start     # runs `npx http-server ./ -c-1`
```

- Or open `plain-index.html` directly in your browser.

## Static HTML/CSS/JS demo

I added a simplified, modern-classic static demo of the site using plain HTML/CSS/JavaScript for easier local preview and sharing:

- `plain-index.html` — homepage (hero, tours, gallery, booking form)
- `plain-kibera.html` — Kibera Street Vibes details page with Basic / Standard / Premium packages
- `styles/static.css` — styling for the static pages
- `scripts/static.js` — simple JS for form interaction and dynamic year

How to view:

- Open `plain-index.html` in your browser (double-click the file or drag it into the browser).
- Or run a lightweight static server (recommended):
  - `npx serve .` or `npx http-server` then open the shown URL.
